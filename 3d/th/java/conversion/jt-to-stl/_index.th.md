JT (Jupiter Tessellation) เป็นรูปแบบข้อมูล 3D ที่มีมาตรฐาน ISO ที่มีประสิทธิภาพ มุ่งเน้นอุตสาหกรรม และมีความยืดหยุ่น พัฒนาโดย Siemens PLM Software โดเมน CAD เครื่องกลของการบินและอวกาศ อุตสาหกรรมยานยนต์ และเครื่องจักรกลหนัก ใช้ JT เป็นรูปแบบการแสดงภาพ 3D ชั้นนำของพวกเขา รูปแบบ JT คือกราฟฉากที่สนับสนุนแอตทริบิวต์และโหนดเฉพาะ CAD เทคนิคการบีบอัดที่ซับซ้อนใช้เพื่อจัดเก็บข้อมูลด้าน (รูปสามเหลี่ยม) รูปแบบนี้มีโครงสร้างเพื่อรองรับแอตทริบิวต์ที่มองเห็นได้ ข้อมูลผลิตภัณฑ์และการผลิต (PMI) และข้อมูลเมตา มีการรองรับที่ดีสำหรับการสตรีมเนื้อหาแบบอะซิงโครนัส ในอุตสาหกรรมเครื่องจักรกลหนัก ผู้เชี่ยวชาญใช้ไฟล์ JT ในโซลูชัน CAD และโปรแกรมซอฟต์แวร์การจัดการวงจรผลิตภัณฑ์ (PLM) เพื่อตรวจสอบรูปทรงของสินค้าที่ซับซ้อน
