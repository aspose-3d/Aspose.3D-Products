        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="DAE" readMoreLink="https://docs.fileformat.com/3d/dae/" >}}

Um arquivo DAE é um formato de arquivo Digital Asset Exchange que é usado para trocar dados entre aplicativos interativos 3D. Este formato de arquivo é baseado no esquema XML COLLADA (COLLAborative Design Activity), que é um esquema XML padrão aberto para a troca de ativos digitais entre aplicativos de software gráfico. Foi adotado pela ISO como uma especificação disponível publicamente, ISO/pAS 17506. DAE arquivos podem ser abertos em aplicativos como Adobe Photoshop, AutoDesk Maya, AutoDesk AutoCAD e APIs como Aspose.CAD.

        {{< /blocks/products/pf/agp/i18n/about-file-text >}}

        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="GLB" readMoreLink="https://docs.fileformat.com/3d/glb/" >}}

GLB é a representação do formato de arquivo binário de 3D modelos salvos no formato de transmissão GL (glTF). Informações sobre 3D modelos, como hierarquia de nós, câmeras, materiais, animações e malhas em formato binário. Este formato binário armazena o glTF ativo (JSON,. Bin e imagens) em um blob binário. Também evita o problema de aumento no tamanho do arquivo que acontece no caso de glTF. O formato de arquivo GLB resulta em tamanhos de arquivo compactos, carregamento rápido, representação completa de 3D cenas e extensibilidade para desenvolvimento posterior. O formato usa modelo/gltf-binário como tipo MIME.
