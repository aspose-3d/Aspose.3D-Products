        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="GLB" readMoreLink="https://docs.fileformat.com/3d/glb/" >}}

GLB é a representação do formato de arquivo binário de 3D modelos salvos no formato de transmissão GL (glTF). Informações sobre 3D modelos, como hierarquia de nós, câmeras, materiais, animações e malhas em formato binário. Este formato binário armazena o glTF ativo (JSON,. Bin e imagens) em um blob binário. Também evita o problema de aumento no tamanho do arquivo que acontece no caso de glTF. O formato de arquivo GLB resulta em tamanhos de arquivo compactos, carregamento rápido, representação completa de 3D cenas e extensibilidade para desenvolvimento posterior. O formato usa modelo/gltf-binário como tipo MIME.

        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="3DS" readMoreLink="https://docs.fileformat.com/3d/3ds/" >}}

Um arquivo com extensão. 3ds representa o formato de arquivo mesh 3D Sudio (DOS) usado pelo Autodesk 3D Studio. Autodesk 3D Studio está no mercado de 3D formato de arquivo desde 1990 e agora evoluiu para 3D Studio MAX para trabalhar com 3D modelagem, animação e renderização. Um arquivo 3DS contém dados para 3D representação de cenas e imagens e é um dos formatos de arquivo populares para importação e exportação de dados 3D. Ele considera informações como locais de câmera, dados de malha, informações de iluminação, configurações de viewport, dados de grupo de suavização, referências de bitmap e atributos para criar vértices e polígonos para renderizar uma cena.
