O formato de arquivo de fabricação aditiva (AMF) define padrões abertos para descrição de objetos para ser utilizado por processos de fabricação aditiva, como 3D Impressão. CAD programas usam o formato de arquivo AMF fazendo uso de informações como geometria, cor e material dos objetos. AMF é diferente do formato STL, pois o lateral não suporta cores, materiais, redes e constelações.

HTML (Hyper Text Markup Language) é a extensão para páginas da web criadas para exibição em navegadores. Conhecida como linguagem da web, HTML evoluiu com requisitos de novos requisitos de informação a serem exibidos como parte de páginas da web. A variante mais recente é conhecida como HTML 5, que oferece muita flexibilidade para trabalhar com o idioma. HTML páginas são recebidas do servidor, onde estão hospedadas, ou podem ser carregadas do sistema local também.
