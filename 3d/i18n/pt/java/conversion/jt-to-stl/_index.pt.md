        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="JT" readMoreLink="https://docs.fileformat.com/3d/jt/" >}}

JT (Jupiter Tessellation) é um formato de dados 3D eficiente, flexível e com foco no setor, padronizado pela ISO desenvolvido pela Siemens PLM Software. Os domínios CAD mecânicos da indústria aeroespacial, automotiva e de equipamentos pesados usam JT como seu formato de visualização 3D mais líder. O formato JT é um gráfico de cena que suporta os atributos e nós que são específicos de CAD. Técnicas de compressão sofisticadas são usadas para armazenar dados de facetas (triângulos). Esse formato é estruturado para oferecer suporte a atributos visuais, informações de produto e fabricação (PMI) e metadados. Há um bom suporte para streaming assíncrono de conteúdo. Na indústria mecânica pesada, os profissionais usam o arquivo JT em suas soluções CAD e programas de software de gerenciamento do ciclo de vida do produto (PLM) para examinar a geometria de mercadorias complicadas.

        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="STL" readMoreLink="https://docs.fileformat.com/cad/stl/" >}}

STL, abreviação de estereolitrografia, é um formato de arquivo intercambiável que representa a geometria tridimensional da superfície. O formato de arquivo encontra seu uso em vários campos, como prototipagem rápida, 3D impressão e fabricação assistida por computador. Representa uma superfície como uma série de pequenos triângulos, conhecidos como facetas, onde cada faceta é descrita por uma direção perpendicular e três pontos representando os vértices do triângulo. Os dados resultantes são usados por aplicativos para determinar a seção transversal da forma 3D a ser construída pelo fabber. Não há informações disponíveis no formato de arquivo STL para representação de cor, textura ou outros atributos de modelo CAD comuns.
