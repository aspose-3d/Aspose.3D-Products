Um arquivo DAE é um formato de arquivo Digital Asset Exchange que é usado para trocar dados entre aplicativos interativos 3D. Este formato de arquivo é baseado no esquema XML COLLADA (COLLAborative Design Activity), que é um esquema XML padrão aberto para a troca de ativos digitais entre aplicativos de software gráfico. Foi adotado pela ISO como uma especificação disponível publicamente, ISO/pAS 17506.

HTML (Hyper Text Markup Language) é a extensão para páginas da web criadas para exibição em navegadores. A variante mais recente é conhecida como HTML 5, que oferece muita flexibilidade para trabalhar com o idioma. HTML páginas são recebidas do servidor, onde estão hospedadas, ou podem ser carregadas do sistema local também. Cada página HTML é composta de HTML elementos como formulários, texto, imagens, animações, links, etc. Esses elementos são representados por tags como img, a, p e vários outros onde cada tag tem início e fim.
