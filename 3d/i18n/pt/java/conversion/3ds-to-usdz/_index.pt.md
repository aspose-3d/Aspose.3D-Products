Um arquivo com extensão. 3ds representa o formato de arquivo mesh 3D Sudio (DOS) usado pelo Autodesk 3D Studio. Autodesk 3D Studio está no mercado de 3D formato de arquivo desde 1990 e agora evoluiu para 3D Studio MAX para trabalhar com 3D modelagem, animação e renderização. Um arquivo 3DS contém dados para 3D representação de cenas e imagens e é um dos formatos de arquivo populares para importação e exportação de dados 3D. Ele considera informações como locais de câmera, dados de malha, informações de iluminação, configurações de viewport, dados de grupo de suavização, referências de bitmap e atributos para criar vértices e polígonos para renderizar uma cena.

Um arquivo com. Usdz é um arquivo não compactado e unencrypetd ZIP para o formato de arquivo USD (Universal Scene Description) que contém e proxies para arquivos de outros formatos (como texturas e animações) incorporado no arquivo e os executa diretamente com o USD tempo de execução, sem qualquer necessidade de descompactação. USDZ arquivos são pacotes cujo design é baseado na nova abstração de nível Ar de um pacote. Usdz foi registrado no IANA e tem o nome do tipo de mídia do modelo e um nome de subtipo de vnd.usd zip e seus detalhes podem ser encontrados na página de registro IANA.
