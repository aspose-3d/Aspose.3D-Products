Um arquivo com extensão 3DS representa o formato de arquivo mesh 3D Studio (DOS) usado pelo Autodesk 3D Studio. Autodesk 3D Studio está no mercado de 3D formato de arquivo desde 1990 e agora evoluiu para 3D Studio MAX para trabalhar com 3D modelagem, animação e renderização. Um arquivo 3DS contém dados para 3D representação de cenas e imagens e é um dos formatos de arquivo populares para importação e exportação de dados 3D. Ele considera informações como locais de câmera, dados de malha, informações de iluminação, configurações de viewport, dados de grupo de suavização, referências de bitmap e atributos para criar vértices e polígonos para renderizar uma cena.

Um arquivo DAE é um formato de arquivo Digital Asset Exchange que é usado para trocar dados entre aplicativos interativos 3D. Este formato de arquivo é baseado no esquema XML COLLADA (COLLAborative Design Activity), que é um esquema XML padrão aberto para a troca de ativos digitais entre aplicativos de software gráfico. Foi adotado pela ISO como uma especificação disponível publicamente, ISO/pAS 17506.
