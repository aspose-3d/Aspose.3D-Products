        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="3MF" readMoreLink="https://docs.fileformat.com/3d/3mf/" >}}

3MF, 3D Format de fabrication, est utilisé par les applications pour rendre 3D modèles d'objets à une variété d'autres applications, plates-formes, services et imprimantes. Il a été conçu pour éviter les limitations et les problèmes dans d'autres formats de fichiers 3D, comme STL, pour travailler avec les dernières versions des imprimantes 3D. 3MF est relativement un nouveau format de fichier qui a été développé et publié par le consortium 3MF. Il est suffisamment riche pour décrire complètement un modèle, en conservant les informations internes, la couleur et d'autres caractéristiques qui le rendent extensible pour prendre en charge de nouvelles innovations dans l'impression 3D. Le format est extensible, peut être largement adopté et exempt de problèmes liés à d'autres formats de fichiers largement utilisés.

        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="USDZ" readMoreLink="https://docs.fileformat.com/3d/usdz/" >}}

Un dossier avec. Usdz est une archive ZIP non compressée et unencrypetd pour le format de fichier USD (Universal Scene Description) qui contient et propose des fichiers d'autres formats (tels que des textures et des animations) intégré dans l'archive et les exécute directement avec le USD running-time sans avoir besoin de déballage. USDZ fichiers sont des paquets dont la conception est basée sur la nouvelle abstraction de niveau Ar d'un package. Usdz a été enregistré auprès de l'IANA et a le nom de type multimédia du modèle et un nom de sous-type de vnd.usd zip et ses détails peuvent être trouvés sur la page d'enregistrement IANA.
