        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="DAE" readMoreLink="https://docs.fileformat.com/3d/dae/" >}}
Un fichier DAE est un format de fichier Digital Asset Exchange utilisé pour échanger des données entre des applications 3D interactives. Ce format de fichier est basé sur le schéma XML COLLADA (COLLAborative Design Activity) qui est un schéma XML standard ouvert pour l'échange d'actifs numériques entre les applications logicielles graphiques. Elle a été adoptée par l'ISO en tant que spécification accessible au public, ISO/pAS 17506. Les fichiers DAE peuvent être ouverts dans des applications telles qu'Adobe Photoshop, AutoDesk Maya, AutoDesk AutoCAD et des API telles que Aspose.CAD.

Développé par Pixar, USD offre la possibilité d'échanger des ressources élémentaires (telles que des modèles) ou des animations.
