Un archivo AMF consta de directrices para la descripción de objetos que se utilizarán en los procesos de fabricación aditiva. Contiene una etiqueta XML de apertura y termina con un elemento. Está precedido por una línea de declaración XML que especifica la versión XML y la codificación del archivo. Las declaraciones pueden incluir información sobre unidades de medida y, en ausencia de dicha información, se utilizan milímetros como unidad predeterminada.

Un archivo con .usdz es un archivo ZIP sin comprimir y sin cifrar para el formato de archivo USD (Descripción de escena universal) que contiene y representa archivos de otros formatos (como texturas y animaciones) incrustados en el archivo y los ejecuta. directamente con el tiempo de ejecución USD sin necesidad de desempaquetar. Los archivos USDZ son paquetes cuyo diseño se basa en la nueva abstracción de nivel Ar de un paquete. Usdz se registró con IANA y tiene un nombre de tipo de medio de modelo y un nombre de subtipo de vnd.usd+zip y sus detalles se pueden encontrar en la página de registro de IANA.
