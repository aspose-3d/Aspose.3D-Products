Un archivo AMF consta de pautas para la descripción de objetos para ser utilizado por los procesos de fabricación aditiva. Contiene una etiqueta XML de apertura y termina con un elemento. Esto viene precedido por una línea de declaración XML que especifica la versión XML y la codificación del archivo. Las declaraciones pueden incluir información de unidades de medida y, en ausencia de dicha información, los milímetros se utilizan como unidad por defecto.

Un archivo con. Usdz es un archivo sin comprimir y unencrypetd ZIP para el formato de archivo USD (Universal Scene Description) que contiene y proxies para archivos de otros formatos (como texturas y animaciones) incrustado en el archivo y los ejecuta directamente con el USD tiempo de ejecución sin necesidad de desempacar. USDZ archivos son paquetes cuyo diseño se basa en la nueva abstracción de nivel de Ar de un paquete. Usdz se registró con IANA y tiene el nombre del modelo de tipo multimedia y un nombre de subtipo de vnd.usd zip y sus detalles se pueden encontrar como en la página de registro de IANA.
