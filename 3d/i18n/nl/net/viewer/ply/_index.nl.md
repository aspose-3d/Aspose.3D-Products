```cs

string output = System.IO.Path.GetTempPath() + Guid.NewGuid().ToString() + ".html";

// Laad PLY scene via een exemplaar van Scene
var scene = new ThreeD.Scene("template.ply");
// Maak een object van Html5SaveOptions en stel eigenschappen in voor het formatteren
var options = new ThreeD.Formats.Html5SaveOptions()
{
    // Zet het raster uit
    ShowGrid = false,
    // Zet de gebruikersinterface uit
    ShowUI = false
};

// Bespaar 3D scene als HTML5
scene.Save(output, options);
// Laad de resulterende HTML in de standaardbrowser
System.Diagnostics.Process.Start(output);


```

PLY, Polygon File Format, vertegenwoordigt 3D bestandsformaat waarin grafische objecten worden opgeslagen die worden beschreven als een verzameling polygonen. Het doel van dit bestandsformaat was om een eenvoudig en gemakkelijk bestandstype tot stand te brengen dat algemeen genoeg is om bruikbaar te zijn voor een breed scala aan modellen. PLY bestandsformaat wordt geleverd als ASCII en binair formaat voor compacte opslag en voor snelle besparing en laden. Het bestandsformaat wordt gebruikt door verschillende applicaties die ondersteuning bieden voor het lezen van 3D bestanden.
