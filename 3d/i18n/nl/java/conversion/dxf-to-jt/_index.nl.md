DXF, Drawing Interchange Format of Drawing Exchange Format is een gelagde gegevensweergave van het AutoCAD-tekenbestand. Elk element in het bestand heeft een geheel getal voor het voorvoegsel dat een groepscode wordt genoemd. Deze groepscode vertegenwoordigt eigenlijk het element dat volgt en geeft de betekenis aan van een gegevenselement voor een bepaald objecttype.
