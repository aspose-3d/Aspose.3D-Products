Portable Document Format (PDF) is een type document dat in de jaren negentig door Adobe is gemaakt. Het doel van dit bestandsformaat was om een standaard te introduceren voor het weergeven van documenten en ander referentiemateriaal in een formaat dat onafhankelijk is van applicatiesoftware, hardware en besturingssysteem. PDF bestanden kunnen worden geopend in Adobe Acrobat Reader/Writer en in de meeste moderne browsers zoals Chrome, Safari en Firefox via extensies/plug-ins.
