Eine Datei mit der Erweiterung .usd ist ein Universal Scene Description-Dateiformat, das Daten zum Zwecke des Datenaustauschs und der Erweiterung zwischen Anwendungen zur Erstellung digitaler Inhalte codiert. USD wurde von Pixar entwickelt und bietet die Möglichkeit, elementare Assets (z. B. Modelle) oder Animationen auszutauschen.

        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="AMF" readMoreLink="https://docs.fileformat.com/3d/amf/" >}}

Eine AMF-Datei besteht aus Richtlinien für die Beschreibung von Objekten, die von additiven Fertigungsprozessen verwendet werden sollen. Es enthält ein öffnendes XML-Tag und endet mit einem Element. Davor steht eine XML-Deklarationszeile, die die XML-Version und Kodierung der Datei angibt. Die Deklarationen können Angaben zu Maßeinheiten enthalten, und wenn solche Angaben fehlen, werden Millimeter als Standardeinheit verwendet.
