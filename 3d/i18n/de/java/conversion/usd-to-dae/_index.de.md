        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="DAE" readMoreLink="https://docs.fileformat.com/3d/dae/" >}}

Eine DAE-Datei ist ein Digital Asset Exchange-Dateiformat, das zum Austausch von Daten zwischen interaktiven 3D Anwendungen verwendet wird. Dieses Dateiformat basiert auf dem XML-Schema COLLADA (COLLAborative Design Activity), einem offenen Standard-XML-Schema für den Austausch digitaler Assets zwischen Grafik software anwendungen. Es wurde von ISO als öffentlich zugängliche Spezifikation ISO/pAS 17506 übernommen. DAE Dateien können in Anwendungen wie Adobe Photoshop, AutoDesk Maya, AutoDesk AutoCAD und APIs wie Aspose.CAD geöffnet werden.
