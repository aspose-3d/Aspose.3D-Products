JT (Jupiter Tes sella tion) ist ein effizientes, branchen orientiertes und flexibles ISO-standard isiertes 3D-Datenformat, das von Siemens PLM Software entwickelt wurde. Mechanische CAD Bereiche Luft-und Raumfahrt, Automobili ndustrie und Schwer maschinen verwenden JT als das führende 3D-Visual isierungs format. Das JT-Format ist ein Szenen diagramm, das die Attribute und Knoten unterstützt, die CAD-spezifisch sind.
