DXF, Drawing Interchange Format oder Drawing Exchange Format ist eine markierte Daten darstellung der AutoCAD-Zeichnung datei. Jedes Element in der Datei hat eine Präfix-Ganzzahl, die als Gruppen code bezeichnet wird. Dieser Gruppen code stellt tatsächlich das folgende Element dar und gibt die Bedeutung eines Daten elements für einen bestimmten Objekt typ an. DXF ermöglicht es, nahezu alle benutzer definierten Informationen in einer Zeichnung datei darzustellen.
