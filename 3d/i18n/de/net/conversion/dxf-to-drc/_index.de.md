        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="DXF" readMoreLink="https://docs.fileformat.com/cad/dxf/" >}}
DXF, Drawing Interchange Format oder Drawing Exchange Format ist eine markierte Daten darstellung der AutoCAD-Zeichnung datei. Jedes Element in der Datei hat eine Präfix-Ganzzahl, die als Gruppen code bezeichnet wird. Dieser Gruppen code stellt tatsächlich das folgende Element dar und gibt die Bedeutung eines Daten elements für einen bestimmten Objekt typ an. DXF ermöglicht es, nahezu alle benutzer definierten Informationen in einer Zeichnung datei darzustellen. Das Dateiformat DXF wurde von Autodesk als CAD Daten dateiformat für die Daten inter operabilität zwischen AutoCAD und anderen Anwendungen entwickelt. Somit können Daten aus anderen Formaten in DXF nach AutoCAD gemäß den Inter operabilität spezifikationen des DXF-Dateiformats importiert werden.

        {{< blocks/products/pf/agp/i18n/about-file-text fileFormat="drc" readMoreLink="https://docs.fileformat.com/3d/drc/" >}}
Eine Akte mit. Die drc-Erweiterung ist ein komprimiertes 3D-Dateiformat, das mit Google Draco Bibliothek erstellt wurde. Google bietet Draco als Open-Source-Bibliothek zum Komprimieren und Dekomprimieren von 3D geometrischen Netzen und Punktwolken und verbessert die Speicherung und Übertragung von 3D Grafiken. Es codiert die Eingabedaten und speichert sie als. Drc-Datei. Am empfangenden Ende liest die API. Drc-Dateien und können diese als PLY-oder OBJ-Dateien ausgeben. Die komprimierte Ausgabe datei ermöglicht es Benutzern, Apps schneller herunter zuladen und 3D Grafiken in Browsern schnell zu laden.
