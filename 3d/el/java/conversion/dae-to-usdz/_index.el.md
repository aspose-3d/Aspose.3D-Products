Αυτή η μορφή αρχείου βασίζεται στο σχήμα XML COLLADA (COLLAborative Design Activity), το οποίο είναι ένα ανοιχτό πρότυπο σχήμα XML για την ανταλλαγή ψηφιακών στοιχείων μεταξύ εφαρμογών λογισμικού γραφικών.
