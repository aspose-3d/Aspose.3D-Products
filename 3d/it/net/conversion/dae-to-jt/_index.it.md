Un file DAE è un formato di file di Digital Asset Exchange utilizzato per lo scambio di dati tra applicazioni interattive 3D. Questo formato di file si basa sullo schema XML COLLADA (COLLAborative Design Activity), che è uno schema XML standard aperto per lo scambio di risorse digitali tra applicazioni software di grafica. È stato adottato da ISO come specifica pubblicamente disponibile, ISO/pAS 17506. DAE I file possono essere aperti in applicazioni come Adobe Photoshop, AutoDesk Maya, AutoDesk AutoCAD e API come Aspose.CAD.
