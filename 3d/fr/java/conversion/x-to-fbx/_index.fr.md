FBX, FilmBox, est un format de fichier populaire 3D initialement développé par Kaydara pour MotionBuilder. Il a été acquis par Autodesk Inc en 2006 et est maintenant l'un des principaux formats d'échange 3D utilisé par de nombreux outils 3D. FBX est disponible au format de fichier binaire et ASCII. Le format a été établi pour assurer l'interopérabilité entre les applications de création de contenu numérique.
